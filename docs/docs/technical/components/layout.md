# Layout

Main layout wrapper providing consistent page structure.

## Features

- Max-width container for content
- Responsive padding
- Background styling
- Centered layout

## Usage

```tsx
import { Layout } from './components/layout/Layout';

<Layout>
  <Header />
  <main>{/* Page content */}</main>
</Layout>
```
