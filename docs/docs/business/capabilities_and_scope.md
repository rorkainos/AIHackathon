# Business Capabilities and Scope

## In-Scope Business Capabilities

### Task Management
- Create new tasks with title, description, and due date/time
- Edit existing tasks (title, description, due date/time)
- Delete tasks
- Mark tasks as complete or incomplete

### Time-Based Organization
- View tasks organized by calendar month
- View tasks in chronological timeline
- See countdown timers for each task showing time until due
- Identify overdue tasks automatically

### Day Closure and Archival
- Complete an entire day, archiving all tasks for that date
- Lock completed days to prevent further modifications
- Maintain a permanent record of which tasks were associated with completed days

### Visual Navigation
- Switch between calendar view and timeline view
- Navigate through months in calendar view
- Jump to current day in both views
- See task counts per day in calendar view

## Out-of-Scope Functionality

The following capabilities are explicitly not provided by the system:

### Multi-User Capabilities
- No user authentication or login
- No user accounts or profiles
- No task sharing or collaboration
- No assignment of tasks to other people

### External Integrations
- No integration with calendars (Google Calendar, Outlook, etc.)
- No email notifications or reminders
- No mobile app synchronization
- No export to other systems

### Advanced Task Features
- No recurring tasks
- No task priorities or categories
- No subtasks or task hierarchies
- No task dependencies
- No attachments or file storage
- No comments or notes beyond the description field

### Reporting and Analytics
- No metrics or statistics
- No productivity reports
- No trend analysis
- No data export functionality

### Team and Enterprise Features
- No shared workspaces
- No approval workflows
- No role-based access controls
- No audit trails for compliance

## Dependencies

### Browser Environment
The system requires a modern web browser with:
- Local storage support for data persistence
- JavaScript enabled
- Date/time picker support

### No External System Dependencies
The application operates independently without requiring:
- Backend servers
- Database systems
- Authentication services
- Third-party APIs
