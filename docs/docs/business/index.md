# Business Documentation

This section contains business documentation for the Todo App.

The Todo App allows users to manage their tasks effectively with features including:

- Creating, reading, updating, and deleting TODOs with due dates and times
- Real-time countdown timers showing time until due
- Completing entire days to lock and archive TODOs
- Local storage persistence in the browser

## Views

The app provides two main views for organizing and displaying TODOs:

- **[Calendar View](views/calendar.md)**: Monthly calendar grid showing TODO counts per day with navigation controls
- **[Timeline View](views/timeline.md)**: Chronological timeline displaying all TODOs grouped by due date

## Available Actions

- **[Create Todo](actions/create-todo.md)** - Process for adding new TODOs
- **[Update Todo](actions/update-todo.md)** - Modifying existing TODOs
- **[Delete Todo](actions/delete-todo.md)** - Removing TODOs permanently
- **[Complete Day](actions/complete-day.md)** - Archiving and locking days
